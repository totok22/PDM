# FSAE低压配电PDM(Power Distribution Module)
OR-RING，电源为DCDC（高压输入，24V输出）和低压电池（7串磷酸铁锂电池）。第一块INA226检测的是汇合后的低压总线侧的电压/流，I2C地址：0x40；第二块INA226检测的是低压电池侧的电压/流，I2C地址：0x41。MCU：STM32F103C8T6.

| **功能模块**       | **引脚 (Pin)**            | **网络标签**         | **连接说明**                                         |
| -------------- | ----------------------- | ---------------- | ------------------------------------------------ |
| **电源**         | 1, 9, 24, 36, 48        | VCC / VDDA       | 连接 3.3V 电源，每个引脚均配有 100nF 去耦电容。                   |
| **时钟 (HSE)**   | 5 (OSC_IN), 6 (OSC_OUT) | OSC_IN / OSC_OUT | 连接 8MHz 无源晶振（额定负载电容12pF)，配两个18pF 负载电容及 1MΩ 反馈电阻。 |
| **调试接口**       | 34 (PA13), 37 (PA14)    | SWDIO / SWDCLK   | SWD 模式，连接调试插针。                                   |
| **复位电路**       | 7 (NRST)                | RESET            | 连接 10kΩ 上拉电阻、100nF 电容及复位按键。                      |
| **启动配置**       | 44 (BOOT0), 20 (PB2)    | BOOT0 / BOOT1    | 均通过 10kΩ 电阻接地，配置为“从主闪存启动”。                       |
| **通信 (UART1)** | 30 (PA9), 31 (PA10)     | **TX / RX**      | 在 MCU 内部，PA9 默认是 TX，PA10 默认是 RX。                 |
| **通信 (CAN)**   | 32 (PA11), 33 (PA12)    | CANRX 、CANTX     | 连接 SN65HVD230 收发器。(波特率500)                       |
| **通信 (I2C)**   | 42 (PB6), 43 (PB7)      | SCL 、SDA         | 连接 4.7kΩ 上拉电阻，用于连接 INA226。                       |
| Alert1         | 11（PA1)                 | ALERT1           | 连接 10kΩ 上拉电阻，连接到第一块INA226的Alert引脚。               |
| Alert2         | 13（PA3）                 | ALERT2           | 连接 10kΩ 上拉电阻，连接到第二块INA226的Alert引脚。               |
| 指示灯 (LED)      | 41（PB5)                 | LED              | 通过 1kΩ 电阻驱动 LED，低电平点亮。                           |


注意：INA226 的 I2C 地址在代码中需左移一位（0x40 << 1 = 0x80）。
